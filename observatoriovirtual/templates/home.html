Plataforma Web para a Execução de Aplicações da área de Astronomia

Publicado Por: INPE


Plataforma Web para execução remota de experimentos com algoritmos para processamento de dados provenientes de simulações no campo da Astronomia.

Atualmente, a plataforma disponibiliza o algoritmo de agrupamento Friends-of-Friends e um algoritmo para remoção de ruído em imagens via transformada Wavelet

Desenvolvido por Otávio Madalosso, Ana Solórzano e Vinícius Monego.
